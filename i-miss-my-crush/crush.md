#Run it with:
#bashpython she_loves_me.py

Features:

Randomly picks a flower with 7, 9, 11, or 13 petals each game
Shows a petal progress bar that empties as you pull
Color-coded output — pink for love, gray for not
Press Enter to pull each petal with a little pulling animation
Dramatic final reveal at the end
Asks to play again when done

Flowers:

Run it with:
bashpython ascii_flower.py           # random flower
python ascii_flower.py rose      # specific flower
python ascii_flower.py daisy
python ascii_flower.py tulip
python ascii_flower.py sunflower
